# PyChess
> A python chess engine

# Running the Engine (Linux)
  1. Setup a Venv on a folder
  2. Download everything onto that folder
  3. Run start.sh
  4. Enjoy!

# Running the Engine (Windows)
  1. Setup a Venv on a folder
  2. Download everything onto that folder
  3. Activate the Venv
  4. Run Chess.py
  5. Enjoy!
