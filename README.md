> A python chess engine

# Running the Engine (Linux)
  1. Install pygame `sudo apt-get install python-pygame`
  2. Download everything.
  3. Run Chess.py
  4. Enjoy!

# Running the Engine (Windows)
  1. Install pygame `pip install pygame`
  2. Download everything.
  3. Run Chess.py
  4. Enjoy!
